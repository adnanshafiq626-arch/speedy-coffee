# Image Requirements for Blog Articles

I cannot generate actual image files, but I've set up the structure and can guide you on what images you need. All images should be placed in: `public/images/articles/`

## Image Specifications

All featured images should be:
- **Format**: JPG (for photos) or PNG (for graphics)
- **Dimensions**: 1200x630px (optimal for SEO and social media sharing)
- **File Size**: Under 200KB (compress using TinyPNG, Squoosh, or similar)
- **Quality**: High resolution, professional-looking
- **Style**: Consistent aesthetic across all images

## Image Optimization

After downloading images, optimize them:

1. **Resize** to exactly 1200x630px
   - Use: Photoshop, GIMP, Canva, or online tools like Squoosh

2. **Compress** to reduce file size
   - Use: TinyPNG.com, Squoosh.app, or ImageOptim

3. **Rename** files exactly as listed above:
   - `perfect-pour.jpg`
   - `sustainable-sourcing.jpg`
   - `coffee-myths.jpg`
   - `cold-brew.jpg`
   - `coffee-tasting.jpg`
   - `grind-sizes.jpg`

4. **Save** to `public/images/articles/` directory
